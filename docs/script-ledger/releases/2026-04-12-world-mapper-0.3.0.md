# Reason World Mapper `0.3.0`

## Date

- 2026-04-12

## Status

- In Progress

## Summary

- added one-click quick survey and export for low-friction nearby area snapshots
- added export summaries so noisy scans can be reviewed faster
- normalized `"null"` and blank object or NPC names into `"Unknown"` in JSON output

## Notes

- the mapper still does not walk by itself in v1; manual route walking and waypoint capture are still the main path for route creation
- quick survey is meant for area snapshots, bank/building scans, and early city mapping where full anchor labeling would be too slow
