# Coordinate Registry

Use this folder to store live in-game coordinates, object IDs, NPC IDs, and route notes gathered during testing.

## Purpose

- keep discovered coordinates out of chat history and in the workspace
- make future bot creation faster
- let both the user and Codex reuse validated route data

## Suggested Format

- one file per server
- group entries by activity, route, or bot
- include:
  - object or NPC name
  - object or NPC id
  - x/y/z coordinates
  - notes about the cluster, route, or interaction
