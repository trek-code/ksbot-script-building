# Handoff Workspace

This folder is for transfer-ready deliverables that can be copied into KSBot for manual testing.

## Standard Package Contents

- final source script if required
- final jar if required
- dependency notes
- runtime config notes
- completed transfer checklist
- linked or copied latest test note

## Folder Use

- `packages/`: staged bot deliverables
- `checklists/`: completed handoff checklists for each transfer
