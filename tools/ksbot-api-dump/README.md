# KSBot API Dump Tool

Generates a local text reference from the installed KSBot API jar.

## Run

```powershell
powershell -ExecutionPolicy Bypass -File "D:\Codex GPT\RSPS\KSBOT Script building\tools\ksbot-api-dump\Export-KsbotApiDump.ps1"
```

## Output

Default output folder:

- `D:\Codex GPT\RSPS\KSBOT Script building\docs\ksbot-api\reference-dump`

This tool is meant to give future agents and developers a local, searchable API surface without depending on the rendered docs website.
